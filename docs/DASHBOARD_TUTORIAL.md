# Dashboard Tutorial - Welcome to Heritage Tracker

**First time here? Don't worry - it's simpler than it looks!**

This guide will help you understand the Dashboard page in 5 minutes. The Dashboard might seem overwhelming at first with all its maps, tables, and controls, but each piece has a simple purpose.

---

## What You're Looking At

The Dashboard is split into **4 main areas**:

```
┌─────────────────────────────────────────────────────────┐
│  1. FILTER BAR (Top)                                    │
│     Search and filter heritage sites                    │
├─────────────────────────────────────────────────────────┤
│  ┌─────────┬─────────────┬──────────────────┐           │
│  │ 2. LIST │ 3. OVERVIEW │ 4. DETAIL VIEW   │           │
│  │  Table  │    Map      │  Satellite Zoom  │           │
│  │         │             │                  │           │
│  └─────────┴─────────────┴──────────────────┘           │
│  ┌─────────────────────────────────────────────────┐    │
│  │  5. TIMELINE (Bottom)                           │    │
│  │     When sites were destroyed                   │    │
│  └─────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────┘
```

Let's explore each area step by step.

## 1. Filter Bar (Top) - Find What You're Looking For

**What it does:** Helps you narrow down the 70 heritage sites to find what you need.

### Quick Start

**Search Box (Left)**
- Type any keyword: site name, location, description
- Results filter instantly (after 300ms)
- Example: Try typing "mosque" or "Gaza"

**Filter Buttons**
- **Type**: Mosque, Church, Archaeological Site, Museum, etc.
- **Status**: Destroyed, Severely Damaged, Partially Damaged, etc.
- **Destruction Date**: Pick a date range (e.g., "Show me sites destroyed in 2023")
- **Year Built**: Filter by construction period (supports ancient dates like "800 BCE")

**How to Use Filters:**
1. Click a button (e.g., "Type")
2. Check boxes in the dropdown (e.g., ☑ Mosque ☑ Church)
3. Click outside to close
4. See results update automatically

**Active Filters**
- Appear as colored pills below the main bar
- Click the ✕ on any pill to remove that filter
- Click "Clear All" to start over

**Results Counter**
- Shows "Showing X of 70 sites" at top left
- Helps you know how many sites match your filters

---

## 2. Sites List (Left Panel) - All Sites at a Glance

**What it does:** Shows all sites that match your filters in a scrollable table.

### Features

**Table Columns:**
- **Name**: Click to open full details
- **Type**: Icon showing mosque, church, etc.
- **Status**: Destruction status with color coding
- **Date Destroyed**: When it was destroyed
- **Year Built**: Original construction date

**Interactions:**
- **Scroll** through the list to see all sites
- **Click any row** → Site highlights on both maps (Overview + Detail View)
- **Click site name** → Opens detailed information modal
- **Click expand icon (⤢)** → Navigate to full Data page with export options

**Resizing:**
- **Drag the handle** on the right edge to make table wider/narrower
- Columns adapt automatically (more columns appear as you widen the table)

---

## 3. Overview Map (Center) - See the Big Picture

**What it does:** Shows all sites on an interactive map with Palestinian flag colors.

### Features

**Map Controls:**
- **Scroll** to zoom in/out (or use Ctrl+Scroll on Windows)
- **Drag** to pan around
- **Click markers** to see site info
- **Street/Satellite toggle** in top-right corner

**Colored Clusters:**
- When many sites are close together, they cluster into circles
- Colors match Palestinian flag (red, black, green)
- Number shows how many sites in that cluster
- Click cluster to zoom in and see individual sites

**Glow Layer:**
- Sites have animated glow effects
- Glow dims as sites are destroyed over time
- Visual representation of cultural loss

**Site Markers:**
- Each marker shows one heritage site
- Click marker → Popup appears with preview
- Click "See More" in popup → Opens full details modal

---

## 4. Detail View (Right Panel) - Satellite Close-Up

**What it does:** Automatically zooms to show the selected site on satellite imagery.

### Features

**Automatic Zoom:**
- When you click a site in the table or map, this panel zooms to that location
- Always shows satellite imagery (not street view)
- Marker appears at the site's exact coordinates

**Historical Imagery Toggle:**
- Switch between 3 time periods:
  - **2014**: Earliest available imagery
  - **Aug 2023**: Mid-point reference
  - **Current**: Most recent satellite imagery (2025)
- Compare how the site looked before and after destruction

**Map Settings (Bottom-Left):**
- **☑ Zoom to Site**: When enabled, map automatically zooms when you select a site
- **☑ Show Map Markers**: Toggle visibility of site markers on satellite view

**Why Two Maps?**
- **Left (Overview)**: See all sites and their spatial relationships
- **Right (Detail View)**: Focus on one site with satellite imagery
- They work together to give you context + detail

---

## 5. Timeline (Bottom) - When Destruction Happened

**What it does:** Shows when each heritage site was destroyed on a timeline.

### Understanding the Timeline

**Visual Elements:**
- **Horizontal line**: Represents time (left = past, right = recent)
- **Year markers**: Labels like "2014", "2020", "2024"
- **Red dots**: Each dot is a destruction event (one site destroyed)
- **Green handle**: Draggable scrubber you can move through time

**Interactions:**
- **Drag the green handle** → Maps update to show which sites existed at that time
- **Click a red dot** → Highlights that site on all maps and in the table
- **Click Play ▶** → Automatically moves through time
- **Adjust speed** → Use dropdown to speed up/slow down (0.5x to 4x)
- **Previous/Next buttons** → Jump between destruction events

**Keyboard Shortcuts:**
- **Space**: Play/Pause
- **Left/Right Arrows**: Move to previous/next event
- **Home**: Jump to start of timeline
- **End**: Jump to most recent

### What's Happening?

As you drag the timeline or press Play:
- Sites appear/disappear based on whether they existed at that time
- Destroyed sites disappear after their destruction date
- Maps update in real-time to show historical state

## Your First 5 Minutes - Try This!

Here's a quick tour to get comfortable:

### Step 1: See All Sites (30 seconds)
- Look at the Overview Map (center)

- Click any row in the Sites List (left)

## Common Questions

### "Why are there two maps?"
- **Overview Map (center)**: See all sites and clusters - answers "Where are sites located?"
- **Detail View (right)**: Zoom to one site with satellite imagery - answers "What does this specific site look like?"

### "What do the colors mean?"
**Map Clusters:**
- Red/Black/Green: Palestinian flag colors
- No status meaning - just for visual identity

**Site Status:**
- Red: Destroyed
- Orange: Severely Damaged
- Yellow: Partially Damaged
- Blue: Looted
- Purple: Threatened

### "How do I see more information about a site?"
Three ways:
1. Click the site **name** in the table (left panel)
2. Click a **map marker** → Click "See More" in popup
3. Click a **red dot** on timeline → Click site name

### "The table is too wide/narrow - how do I resize it?"
- Find the vertical handle on the right edge of the Sites List
- Click and drag left/right
- Columns automatically adjust as you resize

### "Can I search for a specific location?"
- Yes! Use the search box in Filter Bar
- Type city name, neighborhood, or site name
- Results filter instantly

### "What's the difference between 'Year Built' and 'Destruction Date' filters?"
- **Year Built**: When the site was originally constructed (e.g., "800 BCE" or "1277")
- **Destruction Date**: When the site was damaged/destroyed (e.g., "2023-10-15")

### "How do I get back to seeing all sites?"
- Click "Clear All" button in Filter Bar (appears when filters are active)
- Or manually remove all filter pills by clicking their ✕ buttons

### "Can I export the data?"
- Yes! Click the expand icon (⤢) in top-right of Sites List
- This takes you to the Data page
- Export options: CSV, JSON, or GeoJSON

## Tips for Researchers

### Finding Evidence
1. Use **Destruction Date filter** to focus on specific time periods
2. Switch to **Detail View historical imagery** to see before/after
3. Click site name to see **verified sources** (UNESCO, Forensic Architecture, etc.)
4. Export filtered results to CSV for further analysis

### Creating Presentations
1. Filter to your topic (e.g., "Show only mosques destroyed in 2023")
2. Take screenshots of the maps at different time periods
3. Use timeline to show progression of destruction
4. Export data for charts and graphs

### Verifying Information
- Every site includes "Verified By" field
- Click site name → Scroll to "Sources" section
- All sources are linked with dates
- Data comes from UNESCO, Forensic Architecture, Heritage for Peace, etc.
